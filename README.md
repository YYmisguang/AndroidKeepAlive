## 功能

可以提供以下功能：均不依赖任何安卓权限授权
1. 应用保活，拒绝应用强杀，拒绝后台任务栏清理杀死，拒绝清除缓存杀死。
2. 自启动
   1. 首次安装自启动
   2. 固定时间间隔自启动
   3. 安装卸载其他应用自启动
   4. 开关机自启动
3. 后台持续保持活跃，对抗系统休眠。省电策略，可自由选择关闭。
4. 不依赖权限的后台弹框，弹出页面，activity，dialog，成功率达到 90 %。
5. 隐藏后台通知栏。隐藏通知。
